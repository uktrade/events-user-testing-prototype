Create an event

This tool will allow you to create and publish:

  • creation of your event webpage
  • design of your event registration form
  • setup and scheduling of your event emails

Before you begin, it might be useful to familiarise yourself with the structure of the webpage you’ll be populating.

If you’re not yet ready to complete a particular section, that’s fine. Just select the ‘I’ll complete this later’ link at the bottom of each page.

All unpublished events will be saved under the ‘Draft’ tab on your account homepage - so you can keep it as a work in progress until you’re ready to publish.
